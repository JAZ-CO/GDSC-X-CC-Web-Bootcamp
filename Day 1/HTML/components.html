Introduction to HTML

What is HTML?

HTML stands for HyperText Markup Language. It is the standard markup language used for creating web pages.

HTML syntax and structure

HTML documents are structured using tags and elements. Tags are enclosed in angle brackets and elements consist of an opening tag, content, and a closing tag.
